Welcome to the

  Toronto Public Library Programs Manager

Click on one of the tabs above to get started.

Toronto Public Library will be recognized as the world's leading library by informing and inspiring Toronto and its communities, making us all more resilient, more knowledgeable, more connected and more successful.

The Mission of this Web App is to provide information about the Toronto Public Library Attendance rate, a tool to schedule future events, additional analytics to help improve the scheduling planning and also explain the past attendance rates of the Library.

How to Use the Manager

  •   Catalog : Browse all programs, create new programs and create new events
  •   Analytics : Browse analytics of programs and generate visuals based on needs
  •   Text to Speech : Use IBM Watson to generate audio files
